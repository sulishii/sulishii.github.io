## 关于我们

欢迎来到湖南大学设计艺术学院3DXLab实验室！我们是一个专注于计算机视觉、媒体计算、沉浸感知与交互设计、文化遗产数字化的前沿研究团队。

实验室由[劳奕臻教授](https://design.hnu.edu.cn/info/1023/8820.htm)领导，致力于推动三维视觉技术和人工智能在设计艺术领域的创新应用。
